# QAOA Max-Cut with PennyLane

Piccola demo passo passo di QAOA per il problema Max-Cut usando PennyLane.

## Ambiente

```bash
conda env create -f environment.yml
conda activate qaoa-pennylane
```

## Obiettivo

- definire un grafo semplice
- costruire il problema Max-Cut
- implementare QAOA in PennyLane
- analizzare la soluzione trovata
